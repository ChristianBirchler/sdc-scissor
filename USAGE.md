# CPS Sorter

## Usage

### Using with Docker

Requirements: [Docker](https://docs.docker.com/get-docker/)

```bash
docker build --tag cps-sorter .
docker run --rm cps-sorter [OPTIONS]
```

### Directy

Requirements: [Poetry](https://python-poetry.org/docs/)

```bash
poetry install
poetry run python experiments.py [OPTIONS]
```
